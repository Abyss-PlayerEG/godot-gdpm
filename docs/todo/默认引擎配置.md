# 默认引擎配置

## 目标

在 `~/.gdpm/` 下配置默认引擎，支持在非 gdpm 项目中打开 Godot。

## 命令设计

```bash
gdpm godot default <id>           # 设置默认引擎
gdpm godot default                # 显示当前默认引擎
gdpm godot default --unset        # 取消默认引擎
```

## 示例

```bash
# 设置默认引擎
gdpm godot default steam@4.7-stable

# 显示默认引擎
gdpm godot default
# Default engine: steam@4.7-stable

# 取消默认引擎
gdpm godot default --unset

# 在非 gdpm 项目中打开
gdpm godot open
# Opens steam@4.7-stable
```

## 配置文件

`~/.gdpm/local-engines.json`：

```json
{
  "default": "steam@4.7-stable",
  "engines": {
    "steam": {
      "path": "/path/to/Godot.app/Contents/MacOS/Godot",
      "version": "4.7-stable"
    },
    "gdpm-godot": {
      "path": "/path/to/engines/4.7-stable/Godot.app/Contents/MacOS/Godot",
      "version": "4.7-stable"
    }
  }
}
```

## gdpm godot open 行为

| 场景 | 行为 |
|------|------|
| gdpm 项目 + 配置了引擎 | 使用项目配置的引擎 |
| gdpm 项目 + 未配置引擎 | 报错：使用 `gdpm godot use` 配置 |
| 非 gdpm 项目 + 有默认引擎 | 使用默认引擎 |
| 非 gdpm 项目 + 无默认引擎 | 报错：建议配置默认引擎 |

## 错误提示

```
# 非 gdpm 项目 + 无默认引擎
Error: Not a gdpm project and no default engine configured.

  To set a default engine:
    gdpm godot default <id>

  To create a new project:
    gdpm create <name>

  Available engines:
    gdpm godot list -id
```
