# 全局缓存

## 目标

将项目级缓存改为全局缓存，多项目共享，节省磁盘空间。

## 当前状态

```
项目A/.gdpm/cache/   ← 独立缓存
项目B/.gdpm/cache/   ← 独立缓存，重复下载
```

## 目标状态

```
~/.gdpm/
├── cache/
│   ├── downloads/
│   │   ├── c/
│   │   │   └── controller-icons-v2.0.1.zip
│   │   ├── l/
│   │   │   └── limbo-ai-v1.5.0.zip
│   │   └── g/
│   │       └── godot-mcp-native-v1.0.6.zip
│   └── index.json
├── engines/
└── config.toml

项目/
├── gdproject.toml
├── gdpm.lock
└── addons/
```

## 实现步骤

- [ ] 修改 `FileCache` 使用 `~/.gdpm/cache/` 而非项目级目录
- [ ] 修改 `CacheIndex` 存储路径
- [ ] 按首字母分子目录（`c/controller-icons.zip`）
- [ ] 迁移旧缓存（项目级 → 全局）
- [ ] `gdpm cache clean` 清理全局缓存
- [ ] `gdpm cache info` 查看缓存大小

## 对比

| 工具 | 缓存位置 | 级别 |
|------|----------|------|
| npm | `~/.npm/_cacache/` | 全局 |
| pip | `~/.cache/pip/` | 全局 |
| uv | `~/.cache/uv/` | 全局 |
| cargo | `~/.cargo/registry/cache/` | 全局 |
| gdpm（目标） | `~/.gdpm/cache/` | 全局 |
