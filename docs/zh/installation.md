# 安装指南

## pip（推荐）

```bash
pip install godot-gdpm
```

预发布版本需要指定版本号：

```bash
pip install godot-gdpm==0.1.0b12
```

## GitHub Releases

从 [Releases](https://github.com/Abyss-PlayerEG/godot-gdpm/releases) 下载。

=== "macOS / Linux"

    ```bash
    tar -xzf gdpm_*_*.tar.gz
    cd gdpm
    ./install.sh
    ```

=== "Windows"

    解压 zip 并运行 `install.bat`。

## 验证

```bash
gdpm -V
```
