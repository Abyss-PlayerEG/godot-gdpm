# 下载优化

## 目标

优化插件下载速度，支持并行下载和分片下载。

## 当前状态

```
gdpm add: 单插件，单线程下载
gdpm sync: 多插件串行下载
```

## 目标状态

```
gdpm add: 单插件，超过 100MB 自动分片下载（5 片）
gdpm sync: 多插件并行下载（最多 5 个），超过 100MB 自动分片
```

## 实现方案

### 并行下载（gdpm sync）

```python
import asyncio

async def download_parallel(downloads: list, max_concurrent: int = 5):
    semaphore = asyncio.Semaphore(max_concurrent)
    
    async def download_one(item):
        async with semaphore:
            return await item.download()
    
    return await asyncio.gather(*[download_one(d) for d in downloads])
```

- [ ] 获取待下载插件列表
