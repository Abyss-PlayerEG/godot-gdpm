# 笔记

项目开发笔记和待办清单。

> **说明**：本目录包含中文编写的开发笔记。由于 gdpm 是个人项目，主要开发者使用中文，因此笔记使用中文编写以提高迭代速度。官方文档（`docs/` 目录）提供中英双语版本。

## 目录结构

```
notes/
├── done/           ← 已完成功能的设计文档
├── todo/           ← 待开发功能的设计文档
├── todo.md         ← 待办清单总览
└── store-openapi.json  ← Godot Asset Store API 文档
```

## 用途

- 记录功能设计思路和实现方案
- 跟踪开发进度
- 保存参考资料（API 文档等）
